You've enabled Java/JDBC, now what?

Use data.world's JDBC driver to query thousands of datasets on data.world with any program written in a JVM-based language.

Download and configure the data.world JDBC driver.

Download data.world's JDBC driver and create a JDBC connection.

  1. Download the dw-jdbc.jar file directly from the central Maven repository.
    2. Alternatively, you can add the file to a project as a Maven dependency.
  2. Configure the driver in your tool of choice and connect using: 
    3. URL:  jdbc:data:world:sql:[user id]:[dataset id]
User: your data.world username
Password: your data.world API token
    Note: You will need to have your data.world API token, which you can get from data.world's advanced settings.

Start querying your dataset.

After you've connected to data.world with a JDBC connection, you can begin querying the data.

  1. Write & execute your first query.
    2. Example: Using https://data.world/dave/lahman-sabremetrics-dataset.
    SELECT * from halloffame where yearid=? limit 100

What next?

Here are some things you can do with Java/JDBC and data.world:

  • Query datasets in SQL or SPARQL.
    • See more on SQL dialect here.
  • Pull data from data.world into JDBC-compatible tools like RapidMiner, Spark, Drill, TIBCO, and Pentaho to support and enhance ETL & business intelligence implementations.
  • If using Maven, use dw-jdbc by including the following in your pom.xml file: 
    • <dependency>
    <groupId>world.data</groupId>
    <artifactId>dw-jdbc</artifactId>
    <version>0.4.1</version>
<dependency>
